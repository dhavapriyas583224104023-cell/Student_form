# Student Form Webpage

A simple and interactive student form webpage developed using HTML, CSS, and JavaScript.  
This project allows users to enter student details, validates required fields, and displays saved data in a popup table with delete functionality.

Features:
- Student data input form
- Input validation with alert message
- Popup view for saved student records
- Dynamic table rendering using JavaScript
- Clean UI with glass effect design

Technologies Used:
- HTML
- CSS
- JavaScript
